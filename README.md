## Proyectos destacados

### YOKI Marketplace

Aplicación móvil orientada a microempresas y emprendimientos locales de Panamá.

El proyecto busca proporcionar una plataforma organizada para que los emprendedores puedan publicar, vender y gestionar sus productos o servicios. También permite que los usuarios consulten información de los negocios y realicen reseñas.

**Funciones principales:**

- Creación de perfiles para microempresas.
- Publicación y organización de productos y servicios.
- Gestión de categorías.
- Sistema de reseñas y calificaciones.
- Diseño de una interfaz accesible y organizada.

**Tecnologías utilizadas:**

- Java
- XML
- Android Studio
- Firebase
- Git y GitHub

---

### Sistema web para reserva de salón de eventos

Proyecto académico desarrollado para mejorar el proceso de reserva de un salón de eventos del Hotel Azuero.

El sistema busca organizar de manera más clara la disponibilidad de fechas, las solicitudes de clientes y la administración de las reservas.

**Funciones desarrolladas:**

- Diseño de una solución digital para gestionar reservas.
- Organización de fechas y disponibilidad.
- Aplicación de conocimientos de desarrollo web.
- Enfoque en la organización y disponibilidad de la información.

---

### Sitio web para empresa de camisetas

Página web desarrollada para una empresa dedicada a la venta de camisetas.

El objetivo del proyecto fue presentar los productos, la información del negocio y facilitar la promoción digital de la marca.

**Funciones desarrolladas:**

- Creación de una estructura web básica con HTML.
- Presentación de productos y datos comerciales.
- Organización visual del contenido.
- Diseño de una interfaz sencilla para los usuarios.
